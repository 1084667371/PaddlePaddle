# 三、实现思路

#### 使用PaddleX可以省去训练过程的代码，但是数据处理方面的代码是必不可少的，这里就针对数据处理给大家提供一些思路：

* 1. 首先分析拿到的数据集是怎么样的，可以用什么方法解决？（分类、检测、分割等）
* 2. 拿到的图片是否可以直接使用？需要做什么处理才能达到预期效果？
